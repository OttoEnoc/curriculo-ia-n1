# Descrição do Dataset

## Visão Geral
O dataset utilizado neste projeto é composto por 40 perfis curriculares fictícios e anonimizados, construídos pelo grupo para testes do sistema de geração automática de currículos em PDF.

## Origem dos Dados
Os registros foram elaborados manualmente com base em modelos públicos de currículos e adaptados para representar perfis profissionais da área de tecnologia em diferentes níveis de senioridade.

## Estrutura dos Registros
Cada currículo contém, entre outros, os seguintes campos:
- nome
- e-mail
- telefone
- senioridade
- resumo profissional
- experiências profissionais
- formação acadêmica
- habilidades
- idiomas

## Finalidade
O dataset é utilizado para:
1. análise exploratória inicial;
2. preparação e padronização dos dados;
3. testes da futura geração de currículos em PDF com apoio de LLM.

## Privacidade
Todos os dados foram anonimizados/criados de forma fictícia, sem correspondência direta com pessoas reais identificáveis.
