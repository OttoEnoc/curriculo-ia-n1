# Sistema Inteligente para Geração Automática de Currículos em PDF com Uso de IA

Projeto da disciplina de Inteligência Artificial – Universidade Presbiteriana Mackenzie.

## Integrantes
- Otto Enoc Hermano Smeland de Oliveira — RA 10402128
- Eduardo Figueira Losco — RA 10416650

## Descrição
Este projeto propõe o desenvolvimento de um sistema capaz de gerar currículos em PDF a partir de dados fornecidos pelo usuário, utilizando Large Language Models (LLMs) para organizar, estruturar e apresentar as informações de forma clara e profissional.

## Conteúdo do repositório
- `relatorio/`: artigo parcial do projeto (N1) no template SBC.
- `dataset/`: dataset utilizado nos testes, com descrição.
- `notebooks/`: notebook Python com a análise exploratória e seus resultados.
- `src/`: arquivo-fonte da análise exploratória com cabeçalho de identificação e histórico de alterações.
